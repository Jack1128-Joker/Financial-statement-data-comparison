# Theme Name: 点亮
# Vibe & Description: 纯黑背景搭配亮色光影，形成强烈的明暗对比，光线打在利落棱角处，勾勒出轮廓。标题从阴影中浮现，伴随戏剧性的光效。

# Color
- 首先请确在index.css中，default为dark mode
- 背景色：纯黑（#000000）/ 超深灰（#050505）
- 主光源色：冷调、暖调或中性调的光源颜色，营造科技质感，模拟凌厉的聚光效果。
- 点缀色：细腻金属光泽（银 / 亮面铬色），打造光线折射的视觉细节

# Font
- Heading & Body: OPPOSans4 (url: https://resource-static.cdn.bcebos.com/fonts/OPPOSans4.0.woff2)
# Animation
争取实现以下动效：
- 跟随鼠标的光标出现清晰的圆形光效，光源的亮度非常高（手电筒照射质感），带有较小的柔和光晕。聚光效随鼠标移动时，带有轻微顺滑的惯性延迟（弹簧物理动效），提升动态质感。
- 部分隐藏文字初始为纯黑（完全隐形），被聚光光标照射后显现，用户如同用探照灯在深色视觉界面上 “扫描” 信息。
- 元素受光后缓慢淡入，模拟人眼适应黑暗的视觉过程

# Layout
- 元素排布遵循数学精准度，摒弃常规 12 列网格布局。
- 文字即视觉，大字号粗体字与光影形成互动，成为核心视觉元素。

# Elements
- 组件边缘始终保留微弱恒亮光效，即便在暗调中也能清晰勾勒形体。
- 大字号粗重字体，通过mix-blend-mode: overlay（叠加混合模式）或background-clip: text（文字背景裁剪）属性，实现与光线的动态交互效果。
- 中心聚焦视觉：产品 / 主标题为视觉核心主角，其余区域均以暗调留白衬托
标志性设计元素。
- 通过 CSS 径向渐变实现随鼠标追踪的聚光效果，对下层图层形成遮罩 / 显影的视觉互动。
