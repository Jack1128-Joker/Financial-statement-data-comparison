# 需求文档

## 1. 应用概述

### 1.1 应用名称
财务报表比对系统

### 1.2 应用描述
一个面向企业用户的财务报表比对分析网站，支持用户上传或录入不同年份的财务报表数据，通过多年份对比、可视化图表、财务指标分析等功能，帮助用户快速理解公司财务状况变化趋势。

## 2. 用户与使用场景

### 2.1 目标用户
- 企业管理者：需要快速了解公司财务状况
- 财务人员：需要进行财务数据分析与报告
- 投资者/分析师：需要评估企业财务健康度
- 非财务专业人士：需要直观理解财务数据

### 2.2 核心使用场景
- 年度财务数据对比分析
- 多年财务趋势追踪
- 关键财务指标评估
- 财务报告准备

## 3. 页面结构与功能说明

### 3.1 页面结构

```
财务报表比对系统

├── 数据管理
│   ├── 数据录入
│   └── 数据导入
├── 报表对比
│   ├── 资产负债表对比
│   ├── 利润表对比
│   └── 现金流量表对比

│   ├── 核心指标趋势
│   └── 财务比率分析
└── 数据列表
```

### 3.2 功能说明

#### 3.2.1 首页
- 展示系统功能介绍
- 提供快速入口：数据录入、数据导入、报表对比、指标分析
- 展示已录入的年份数据概览

#### 3.2.2 数据管理

**数据录入页面**
- 选择年份（年度选择器）
- 选择报表类型：资产负债表、利润表、现金流量表
- 提供表单录入界面，包含各报表的标准科目字段
- 资产负债表科目：流动资产、非流动资产、流动负债、非流动负债、所有者权益等
- 利润表科目：营业收入、营业成本、营业利润、净利润等
- 现金流量表科目：经营活动现金流、投资活动现金流、筹资活动现金流等
- 保存数据到后端

**数据导入页面**
- 上传Excel或CSV文件
- 文件需包含年份、报表类型、科目名称、金额等字段
- 系统解析文件内容并展示预览
- 确认导入后保存数据到后端

#### 3.2.3 报表对比

**资产负债表对比页面**
- 选择需要对比的年份（支持多选，至少2个年份）
- 以表格形式横向展示多年份资产负债表数据
- 表格列：科目名称、年份1金额、年份2金额、年份3金额...
- 自动计算同比增长率（相邻年份对比）
- 自动计算环比增长率（与上一年份对比）
- 提供导出按钮，支持将当前对比结果导出为Excel或CSV文件

**利润表对比页面**
- 选择需要对比的年份（支持多选，至少2个年份）
- 以表格形式横向展示多年份利润表数据
- 表格列：科目名称、年份1金额、年份2金额、年份3金额...
- 自动计算同比增长率
- 自动计算环比增长率
- 提供导出按钮，支持将当前对比结果导出为Excel或CSV文件

**现金流量表对比页面**
- 选择需要对比的年份（支持多选，至少2个年份）
- 以表格形式横向展示多年份现金流量表数据
- 表格列：科目名称、年份1金额、年份2金额、年份3金额...
- 自动计算同比增长率
- 自动计算环比增长率
- 提供导出按钮，支持将当前对比结果导出为Excel或CSV文件

#### 3.2.4 指标分析

**核心指标趋势页面**
- 选择需要分析的年份范围
- 以折线图展示关键指标的多年趋势：
  - 营业收入趋势
  - 净利润趋势
  - 净资产趋势
  - 经营活动现金流趋势
- 每个指标图表下方展示具体数值与同比增长率
- 提供导出按钮，支持将当前指标趋势数据导出为Excel或CSV文件

**财务比率分析页面**
- 选择需要分析的年份范围
- 自动计算并展示关键财务指标：
  - 毛利率 = (营业收入 - 营业成本) / 营业收入
  - 净利率 = 净利润 / 营业收入
  - ROE（净资产收益率） = 净利润 / 净资产
  - 资产负债率 = 总负债 / 总资产
- 以柱状图或折线图展示各指标的多年对比
- 展示各指标的同比变化情况
- 提供导出按钮，支持将当前财务比率分析结果导出为Excel或CSV文件

#### 3.2.5 数据列表
- 展示已录入的所有年份数据
- 列表字段：年份、报表类型、录入时间
- 支持编辑已录入数据
- 支持删除已录入数据
- 提供导出按钮，支持将已录入的财务报表数据导出为Excel或CSV文件

## 4. 业务规则与逻辑

### 4.1 数据录入规则
- 同一年份的同一报表类型只能录入一次，重复录入时覆盖原数据
- 金额字段仅支持数字输入，单位为元
- 年份范围限制为1900年至当前年份

### 4.2 数据导入规则
- 支持Excel（.xlsx、.xls）和CSV（.csv）格式
- 文件需包含必填字段：年份、报表类型、科目名称、金额
- 导入时自动校验数据格式，格式错误时提示用户

### 4.3 对比分析规则
- 同比增长率计算公式：(本年数据 - 上年数据) / 上年数据 × 100%
- 环比增长率计算公式：(本期数据 - 上期数据) / 上期数据 × 100%
- 当上年或上期数据为0时，增长率显示为「-」
- 增长率保留2位小数

### 4.4 财务指标计算规则
- 毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
- 净利率 = 净利润 / 营业收入 × 100%
- ROE = 净利润 / 净资产 × 100%
- 资产负债率 = 总负债 / 总资产 × 100%
- 当分母为0时，指标显示为「-」
- 指标保留2位小数

### 4.5 数据存储规则
- 所有录入和导入的数据存储到后端数据库
- 数据按年份、报表类型、科目进行组织
- 支持数据的增删改查操作

### 4.6 数据导出规则
- 支持导出格式：Excel（.xlsx）、CSV（.csv）
- 导出内容包含当前页面展示的所有数据
- 导出文件命名规则：功能名称_日期时间（如：利润表对比_20260717.xlsx）
- 导出的Excel文件保留表格格式与计算结果
- 导出的CSV文件以逗号分隔，编码为UTF-8

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户未录入任何数据时访问对比页面 | 提示「暂无数据，请先录入或导入财务报表数据」 |
| 用户仅录入1个年份数据时尝试对比 | 提示「至少需要2个年份的数据才能进行对比」 |
| 导入文件格式错误 | 提示「文件格式不正确，请上传Excel或CSV文件」 |
| 导入文件缺少必填字段 | 提示「文件缺少必填字段：年份/报表类型/科目名称/金额」 |
| 导入文件中金额字段非数字 | 提示「金额字段必须为数字,请检查文件内容」 |
| 计算增长率时上年数据为0 | 增长率显示为「-」 |
| 计算财务指标时分母为0 | 指标显示为「-」 |
| 用户选择的年份范围内缺少某年数据 | 图表中该年数据点显示为空或断点 |
| 用户在无数据页面点击导出 | 提示「暂无数据可导出」 |

## 6. 验收标准

1. 用户进入数据录入页面，选择年份2025，选择报表类型为利润表，录入营业收入、营业成本、净利润等科目数据，点击保存成功
2. 用户进入数据录入页面，选择年份2026，选择报表类型为利润表，录入营业收入、营业成本、净利润等科目数据，点击保存成功
3. 用户进入利润表对比页面，选择年份2025和2026，系统展示两年利润表数据的横向对比表格，并自动计算同比增长率
4. 用户点击导出按钮，选择Excel格式，系统生成并下载利润表对比_20260717.xlsx文件，文件包含完整对比数据与增长率
5. 用户进入核心指标趋势页面，选择年份范围2025-2026，系统展示营业收入、净利润的折线图趋势
6. 用户进入财务比率分析页面，选择年份范围2025-2026，系统自动计算并展示毛利率、净利率、ROE、资产负债率的对比图表
